# Changelog

All notable changes to the "Vue HTML Bridge" extension will be documented in this file.

## [0.0.1] - 2025-01-27

### Added

- Initial release
- Real-time HTML validation for Vue SFC files
- Integration with markuplint for HTML linting
- LSP-based diagnostics with inline error reporting
- Cartesian coverage: validates all template permutations (v-if/v-else, v-show, v-for)
